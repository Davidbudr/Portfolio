RPG System v1
(Dialogue Tree System)

Concept:

My goal with this project was to make a system to make it easier for me to build rpg games later on and to achieve this I created a custom Unity editor.

Overview:

This was my first attempt at an RPG system, as such I designed it as a node style system similar to blenders nodes or unity's shader graph extensions as I believed that this design would give a clear visual indication of how the player could interact with the world.

While this project was not made for any perticular game it was meant to be both a proof of concept for myself and a stepping stone to creating a game with a branching narritive, however once I completed this project I moved on to working different concepts and left the project on hold for some time.

Unfortunately upon coming back to the system, I realised that while it was visually appealing, its usability was somewhat lacking and some features which I believed would be helpful were more so a distraction, and while building an RPG I realised that certain very useful features were lacking (i.e. events caused by player choices, e.g. the player choosing the correct answer to a riddle opening a door within the game) as such I decided to build the RPG System v2.
