Database Editor

Concept:

My goal was to create a system where new 'pets' and 'furniture' could be added easily to the game, even by an absolute beginner to Unity.

Overview:

Unlike many of my other 'tools' for unity, this Database Editor was made out of â€‹necessity. My client for the 'Little Clickers' project had little to no knowledge of how to use unity, and something that they wanted for the project was to be able to add new pets and furniture items to the game. 

Due to this I had a choice of either teaching the client to use the existing inspector to modify the large list of pets,furniture,clothes, and backgrounds for the game, or alternatively create a more user friendly editor which would allow the client to easily add or remove objects as well as showing only the information that would be relevant to the process.

I chose to do the latter, and as such I created a system where the 4 object types are split into their own tabs and within those tabs it would be possible to explore each item, modifying it with only the most important information displayed to the end user.
