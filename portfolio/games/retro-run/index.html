Retro Run

You can play Retro Run HERE

” Whoops! You’re going the wrong way on the Moonlight super highway, duck and weave through oncoming ships and keep your shield up with the FutureTech™ Spacers-Battery, and if you already have a shield? Worry not, you can sell all unused Spacers-Batteries for some sweet, sweet points. ”

Retro Run is a very simple and short infinite runner based on a small programming test that I had previously completed. The idea behind the game is to be relaxing but have rising tension as the game goes on.

Prev
Next

This is a simple game which I didnt want to over complicate, however I did add a few things to up the challenge of the game, including varience in speed of the oncoming ships as well as those ships being able to avoid objects in front of themselves.

The game is heavily inspired by the Synthwave aesthetic found in several youtube videos titled “synthwave” or “chillwave” in preticular the Videos “Graviton” by Glitch Tape and “Space Trip” by Asthenic which themselves borrow their visuals from 80’s/90’s Anime and Movies such as Drive.

The game Outrun was also a big inspiration for this game as I felt that the camera angles and somewhat relaxing atmosphere of that game gave the same feeling that I was going for, because of that I named the game Retro run as a sort of homage.
